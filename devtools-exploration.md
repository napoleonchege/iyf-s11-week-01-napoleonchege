## Website 2: developer.mozilla.org

### 1. Find the navigation menu - what tag is it wrapped in?
The navigation menu is wrapped in a `<nav>` element.

### 2. How is the search bar structured?
The search bar is inside the navigation section and uses a search component with a search input field.

### 3. What happens when you hover over links?
When hovering over links, the cursor changes to a pointer and the link styling changes, including underline and color effects.


## Website 3: GitHub

### 1. Identify 5 different HTML elements
- `<header>` – contains GitHub top navigation bar
- `<nav>` – holds site navigation links
- `<main>` – contains page content
- `<button>` – used for interactive actions (login, search)
- `<form>` – used for search/login input areas

### 2. Find a form element and list its inputs
Search form:
- Search input
